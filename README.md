# hwFive

### Created on 6/15/2020

### By Sam Ascheim

## Description:

This is a day planner in which the line item for each hour will show grey upon start of the hour, and the inputs will be saved to user's local storage when the button "Save" is clicked.

## Set-up & Installation requirements:

None needed besides an up-to-date web browser.
